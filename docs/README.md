# OHAENG(五行戰記) - Zero Point 문서

카카오톡 단체방 기반의 **텍스트 RPG 서버 개인 프로젝트**입니다.  
채팅 명령 기반 요청 처리 구조를 직접 설계하면서, **요청 정규화, 멱등성, 잠금 전략, 트랜잭션, 상태 전이** 같은 실전형 백엔드 설계 문제를 다루기 위해 만들었습니다.

### 핵심 포인트
- 채팅 명령 기반 요청 처리 파이프라인 설계
- 멱등성, 잠금, 트랜잭션을 고려한 처리 흐름 구조화
- PvP / DOWN / 레이드 규칙을 UseCase 중심으로 분리
- 상세 설계와 정책은 Wiki, 저장소 문서는 1장 요약 중심으로 관리

> 원칙: **Repo docs는 1장 요약만**, 상세 설계/정책은 **GitHub Wiki**로 관리합니다.  
> `docs/`에는 최종적으로 **README.md + ARCHITECTURE_V0.md만** 커밋합니다.
